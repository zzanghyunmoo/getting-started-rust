# 제1주 강의 계획서

## 강의 개요

**주제**: 프로그래밍과 컴퓨팅 개론 - 첫 번째 Rust 프로그램
**목표**: 컴퓨터 과학의 기본 개념을 이해하고, Rust 개발 환경을 설정하여 게임 제목 화면을 출력
**예상 소요 시간**: 2-3시간

---

## 학습 목표

### 지식 목표
- 컴퓨터 과학의 정의와 핵심 분야 이해
- 프로그래밍 언어의 진화 과정과 분류 방법 습득
- Rust 언어의 특징과 장점 파악
- 컴파일러와 인터프리터의 차이 이해

### 기술 목표
- Rust 개발 환경(rustup, cargo) 설치 및 설정
- VS Code에서 Rust 개발 환경 구성
- Cargo 프로젝트 생성 및 실행
- println! 매크로를 사용한 텍스트 출력

### 실습 목표
- Hello World 프로그램 작성 및 실행
- println! 매크로의 다양한 포맷팅 기능 활용
- 게임 제목 화면 구현
